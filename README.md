# The Social Warehouse

This is a data warehouse and data lake system for social, civic and social analysis from [Siege Analytics](1).

Built with:

- [Kubernetes](4)
- [MinIO](13)

Runs:  

- [PostgreSQL](5) + [PostGIS](6)
- [Python](7)
- [R](8)
- [Geoserver](9)
- [Zeppelin Notebook](10) 
- [GeoTrellis](11)
- [Ubuntu](12)
